# Cafe POS / Billing System ☕

A cross-platform desktop Point of Sale (POS) application built with Python and Tkinter for managing café orders and printing receipts using thermal printers (ESC/POS).

The system allows café staff to quickly create orders, calculate totals, and generate professional receipts with automatic hardware detection for USB and Bluetooth thermal printers.

Technologies: Python, Tkinter, ESC/POS, PyInstaller
